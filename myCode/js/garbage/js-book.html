• Array - Массив пронумерованных элементов, также может служить стеком или очередью

      • Создание
      • var arr = []

      Метод
      • push - Добовление элементов в конец массива.

          Синтакс: array.push( elem1, elem2, ... )

          Пример №1:
          // array.length = 2
var array = [ "one", "two" ]  

// добавить элементы "three", "four"
var pushed = array.push("three", "four") 

// теперь array = [ "one", "two", "three", "four" ] 
// array.length = 4
// pushed = 4
